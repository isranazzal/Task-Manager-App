# Task Manager App

## Description

Task Manager is a simple and interactive To-Do List web application built using **HTML5**, **CSS3**, and **JavaScript**.

The application allows users to add and manage tasks through a clean interface. This project focuses on practicing JavaScript DOM manipulation, event handling, and creating dynamic web page interactions.

---

## Technologies Used

- HTML5
- CSS3
- JavaScript

---

## Features

- Add new tasks
- Dynamic task list creation
- User input handling
- Interactive task management interface
- Clean and simple design
- Responsive layout
